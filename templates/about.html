Project Overview

PowerFault AI is a machine learning–powered web application that detects and classifies power system faults in real time. It accepts 10 electrical sensor measurements as input, derives 5 additional physics-based features, and classifies the power system condition into one of 5 categories using a trained ensemble classifier.

⚡ Technical Stack
  • Python 3.8+ / Flask 3.x backend
  • Scikit-learn + XGBoost ML pipeline
  • Bootstrap 5 + Chart.js frontend
  • Joblib model serialization
  • REST API with JSON responses
🎯 Key Features
  • Multi-model training & automatic best-model selection
  • Confidence-based uncertainty detection
  • Root cause + maintenance recommendations
  • Interactive analytics dashboard
  • Dark mode + full mobile responsiveness
